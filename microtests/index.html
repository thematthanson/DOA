Purpose: This microtests suite provides isolated testing environments for each major component of the Rumi system. Each test demonstrates specific functionality and can be run independently to verify system behavior.

How to Use This Suite:

  • Individual Testing: Click any test to open it in a new tab and verify specific functionality
  • Component Isolation: Each test runs independently without dependencies on other system parts
  • Debug Information: Open browser console (F12) to see detailed logging and error messages
  • Visual Feedback: All tests provide real-time visual feedback and state changes
  • Cross-Browser: Tests are designed to work across different browsers and devices

What Each Test Demonstrates:

  • Visual Components: ASCII art, progress bars, animations, and UI elements
  • Data Processing: CSV parsing, content management, and intelligence classification
  • System Logic: Points calculation, multiplier progression, and session management
  • Communication: Iframe messaging, event handling, and cross-component data flow
  • User Interface: Popups, transitions, state management, and responsive design
